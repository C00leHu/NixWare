External CS2 Cheat written in Python3

## Installation

```
pip install -r requirements.txt
```
> PyMeow is also a dependency, install instructions are [here](https://github.com/qb-0/pyMeow?tab=readme-ov-file#floppy_disk-installation)

## Usage

```
python main.py
```
